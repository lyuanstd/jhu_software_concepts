GradCafe PostgreSQL Analysis

This page displays SQL analysis results from the GradCafe applicant database.

Data Controls

Pull Data: This button runs the Module 2 scrape and clean code to check GradCafe for new entries. If new data is available, the local PostgreSQL database is updated.

Note: This process may take some time because it opens a browser, checks recent GradCafe pages, cleans the scraped data, and inserts new records into PostgreSQL.

Update Analysis: The top right button refreshes the analysis using the most recent data currently stored in the database. If a request to pull data is currently running, the analysis will not update.
